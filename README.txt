This Project was done by Islomzhan Akhmedov and Abdullah Alsayari

Boolean circuit simulator written in C.
Part A {[X AND (NOT Y)] OR [Y AND Z]}
Part B {[X NAND (NOT Y)] NOR [Y NAND Z]}
Part C {[Y AND X] OR [(NOT X) AND (NOT Y)]}
The diagrams will be added later. 

HOW TO RUN:
type “make” to build the project
type “./main” to test
